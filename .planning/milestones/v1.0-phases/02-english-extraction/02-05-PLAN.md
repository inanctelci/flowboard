<headless_rule_reminder>
Every file in this plan EXCEPT `App.tsx` is a `.ts` (non-component) file. They MUST use the headless singleton:

```ts
import i18n from "../i18n/i18n";  // path relative to file location
const msg = i18n.t("gen.no_board_loaded");
```

`useTranslation()` in a `.ts` file is a Rules of Hooks violation. The typed-key TS gate will NOT catch it, but it WILL runtime-throw.

`App.tsx` is the exception — it already has `const { t, i18n } = useTranslation()` from Phase 1 (BUGS-03 wired the lang sync effect). Reuse the existing hook for the `"Loading board…"` extraction; do NOT add a second hook.

`constants/character.ts` is a special case — see Task 2 action.
</headless_rule_reminder>
